# Galotxas

Plataforma web para la gestión y visualización de competiciones de Galotxas.

## Estructura inicial

- `backend/`: API y lógica de negocio
- `frontend/`: interfaz web
- `docker/`: infraestructura de desarrollo
- `docs/`: documentación técnica y funcional

## Objetivo

Construir una aplicación desacoplada con backend reutilizable, preparada para web pública, gestión interna y posible app móvil futura.
